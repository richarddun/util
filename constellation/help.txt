Constellation Navigation / Use Help

Up arrow / Down arrow - Navigate through the Counter / Devices
Right arrow / Enter - Jump from Counter to Device select (into submenu)
Left arrow / Backspace - Jump back from Device to Counter (into main menu)
Space key - Select or De-select a Device (in submenu)
Tab key - Jump to or from Time select menu
F12 key - Compute and display device and rate
F10 key - Back to counter select menu
'H' - Show or hide this help message
